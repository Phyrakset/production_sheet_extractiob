# Separated Components Structure

This document outlines the structure of the `test_data_and_knowledge_base/separated_components` directory. This directory contains sub-folders categorized by **Style Number**, and within each folder, the production documents are split into logical, separated PDF components based on their content.

## General Component Naming Convention

The files follow a strict naming convention:
`Comp_[Number]_[Chinese_Name]_[English_Name]_Doc_[Style_Number]_Pages_[Page_Numbers].pdf`

**Important Note on Component Numbers**: There are 19 total standardized components in this system. The `Comp_[Number]` prefix directly corresponds to its numerical position within these 19 components. For example, `Comp_06` is used for the 6th component (Colorways).

Based on this convention, here are the general purposes of each of the 19 component file types:

- **Comp_01 (注意大點 / Key Notes)**: High-level critical information, warnings, or general instructions for the style.
- **Comp_02 (订单明细 / Order Details)**: Order quantities, purchase orders (POs), shipping destinations, and delivery dates.
- **Comp_03 (款式图 / Tech Sketch)**: Technical drawings and design illustrations of the garment.
- **Comp_04 (生产工艺 / Construction)**: Specific manufacturing methods, stitching details, and garment construction techniques.
- **Comp_05 (缝制标准 / Mfg Standards)**: Detailed sewing requirements, stitch types (e.g., SPI - Stitches Per Inch), and seam allowances.
- **Comp_06 (颜色 / Colorways)**: Color palettes, color combinations, and color matching instructions for the style.
- **Comp_07 (面料物料 / BOM Fabrics)**: Bill of Materials specifically detailing the main fabrics and shell materials.
- **Comp_08 (辅料 / BOM Trims)**: Bill of Materials detailing trims (buttons, zippers, threads, elastic, etc.).
- **Comp_09 (唛头标签 / Labels)**: Placement, design, and content of brand, size, care, and composition labels.
- **Comp_10 (印花绣花 / Artwork)**: Details and placements for prints, embroidery, and other graphical embellishments.
- **Comp_11 (成品尺寸 / POMs)**: Points of Measure - specific measurements required for the finished garment across various sizes.
- **Comp_12 (放码规则 / Grading)**: The rules for scaling measurements across different sizes (e.g., from S to M, M to L).
- **Comp_13 (度尺图 / HTM Guide)**: "How to Measure" visual guide showing where to place the measuring tape for each POM.
- **Comp_14 (量度QA / Measure QA)**: Quality Assurance specifically related to measurement tolerances and verification.
- **Comp_15 (PP办评语 / PP Comments)**: Pre-Production sample feedback and corrections to be applied to bulk production.
- **Comp_16 (实物照片 / Fit Photos)**: Photographs of the garment on a fit model or mannequin for visual reference.
- **Comp_17 (质量标准 / QA Standards)**: General Quality Assurance requirements, testing standards, and acceptable defect limits.
- **Comp_18 (包装出货 / Packaging)**: Folding methods, polybag specifications, carton packing instructions, and shipping marks.
- **Comp_19 (修改记录 / Revision History)**: Log of changes made to the document or style over time.
